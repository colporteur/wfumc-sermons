# WFUMC Sermon Archive

Companion app to the WFUMC Bulletin app. Lists every sermon you've
preached, with full-text search across title, scripture, theme, notes,
and manuscript.

Sermons are created automatically when you fill in a sermon item in
the bulletin app — this app doesn't create new sermons, only browses,
searches, and edits metadata for existing ones.

**Shares the bulletin app's Supabase backend.** Same staff login.

See `docs/SETUP.md` for one-time setup steps.
